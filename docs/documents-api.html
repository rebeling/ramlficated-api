Query Parameters

  • q: (string - default: *:* - minLength: 3)

    Search query term. Search is performed on the document body and title.

  • sort: (string - default: relevance)

    Search results are sorted by their relevance to the query term (q). Use the sort parameter to sort by date.

  • page: (number)

    The page to return

  • page_size: (number - default: 10)

    The number of items per page
