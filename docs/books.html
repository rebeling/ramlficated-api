post /books

Add a book to database.

  • Request
  • Response

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/draft-03/schema",
  "type": "object",
  "properties": {
    "id": {
      "type": "string",
      "required": true
    },
    "title": {
      "type": "string",
      "required": true
    },
    "isbn":  {
      "type": "string"
    },
    "authours": {
      "type": "array",
      "item_type": "object",
      "uniqueItems": true,
      "minItems": 1
    }
  }
}

Example:

{
 "id": "53831",
 "title": "Deception Point",
 "isbn": "0-671-02738-7",
 "authors": [
  "Dan Brown"
 ]
}

HTTP status code 201

Book has been successfully created.

HTTP status code 409

Book already exists.

get /books/{id}

Retrieve book-related information.

  • Request
  • Response

URI Parameters

  • id: required (string)

    Book Identifier

HTTP status code 200

Body

Type: application/json

Example:

{
 "id": "53831",
 "title": "Deception Point",
 "isbn": "0-671-02738-7",
 "authors": [
  "Dan Brown"
 ]
}
